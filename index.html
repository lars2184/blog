May 9, 2015

Advanced Front End Development: First Week

I’m taking a Front End Development course through Portland Code School and the first week is in the books. Overall, it was a good week although I have some catching up to do because I jumped straight into the advanced class! I need to brush off my GitHub skills, get up to speed with Treehouse and figure out what the heck Slack is.

In the first class, we were introduced to pair programming, which is something I’ve never done. As an introduction, we played a game called Free Cell in pairs. We were then given an assignment to create a web site for a fictional company called the Animal Adoption Agency. Stephanie and I paired up over Skype to work on the project and it worked well sharing windows so we could work on the same code. We were able to store the list of kittens in a JSON object and use the list to display them.

In the second class, we got a review of jQuery. It’s been awhile since I’ve used jQuery, so it was good to go over some key methods. We also had a good overview of the following tabs in the Chrome Developer Tools console:

Network

I never knew how to make this tab work, but the trick is to click the ‘Disable cache’ box and then refresh the page. All of the page assets are then shown along with load times. A blue line shows when the page is loaded and a red line shows when it is ready, which corresponds to the ready jQuery event.

Sources

This is where you can debug files. Once you type command + p to select a file, you can set breakpoints in the code where the code execution should stop. Once stopped, you can inspect the objects and variables to debug the code. To advance the code, click the ‘step over’ button or click play to resume. If there’s an object or variable you’d like to keep an eye on you can enter it in the ‘Watch Expressions’ panel.
